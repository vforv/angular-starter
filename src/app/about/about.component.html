Angular Starter provide all compilation options, server side rendering, dev environment, testing.

Development

To start dev server run: npm run watch after compiling finish run npm run serve it will start app on port 8000. On each code change server will recompile and you will get get notification

Google closure compiler production build

Google closure require java to run. Configuration file is in root closure.conf. To build with colosure type: npm run build:closure

Webpack production build

There is also option to compile with webpack npm run build:prod Compliling with Google closure result in much better performance of application.

Testing

To start tests run npm run test.
